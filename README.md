## Веб приложение "CRM"

### Установка и запуск (Windows)

- Установите Docker Desktop
- Установите Git
- Клонируйте проект себе в папку с репозитория по [адресу](https://github.com/DEV-WWW-26/crm.git) запустив команду: 
    > git clone https://github.com/DEV-WWW-26/crm.git
- запустите контейнеры запустив исполняемый файл в корне директории: 
    > ./start.bat
  - запустите веб [ссылку](http://localhost)
